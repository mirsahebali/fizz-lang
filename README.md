# Fizz Programming Langauge
- Written in Pure C99
- No external dependencies(except standard library)
- Custom string library functions
- Simplistic programming semantics
- Supports first class functions

## Requirements:

- GCC compiler, supports std=C99
- make >= 3.0
- That's it


## Build binary
```bash
make build

```

## Run tests

```bash

make check test

```

## Run repl

```bash
 make run
 ```
